## Description
#### **One Shot Sample Pack + Project for Elektron Analog Rytm**
This collection of samples was created with Doepfer Dark Energy MK1 and Elektron Analog Four MK1(Filter, Noise, CV Control). 
This pack contains semi-realistic FM / AM percussion and a number of Dark Energy Mk1 waveforms. (triangle, square, sawtooth, Filter oscillation, etc).

### Included

- 838 samples, 16bit + 24bit / 48kHz / Wav
    - BD -115, Snare -101, Rim Shot -35, Clap -63, Cymbal -30, Ride -30,  Hihat -54, Percussion -215 , Tambourine -19, Melodic Sounds[ Chord -30, Bass -10, Lead -21, Wave-115] -176
    - Single Cycle Waveforms - 62[16bit/48kHz/WAV + 16bit/44.1kHz/WAV ]-Update(February 4/2019)
- 1 Analog Rytm Project File
    - 7 Kits, 7 Patterns
